# Elektromagnetische Visualisatie met micro:bit

Dit project is bedoeld om magnetisme te visualiseren met behulp van een micro:bit en een webapplicatie. De micro:bit meet magnetische velden en stuurt deze gegevens naar de webapplicatie, waar ze worden weergegeven in een interactieve visualisatie.

## Benodigdheden
- Een micro:bit met de juiste firmware geïnstalleerd.
- Een computer met een webbrowser die Web Serial API ondersteunt (zoals Google Chrome).
- Een USB-kabel om de micro:bit aan te sluiten op de computer.

## Installatie

1. Upload de micro:bit code naar je micro:bit. Je kunt de code vinden in de `microbit` map van dit project.
2. Na het uploaden, moet je de micro:bit kalibreren door deze in verschillende oriëntaties te plaatsen en de instructies op het scherm te volgen.
2. Ga naar de [GitHub-pagina](microbit_magnetisme) van het project.

## Gebruik

1. Select "Connect to micro:bit" om verbinding te maken met je micro:bit.
2. Zorg ervoor dat de micro:bit op een stabiele ondergrond ligt voor nauwkeurige metingen.
3. Bekijk de visualisatie van het magnetische veld op het scherm.
4. Kalibreer de micro:bit indien nodig door op de knop "Calibrate earth's magnetic field" te klikken.
5. Bekijk de gegevens en visualisaties die worden weergegeven op het scherm. Onderzoek de invloed van het elektromagnetisch veld veroorzaakt door de elektrische stroom door een draad of spoel.
